Drift

This summarizes the results of instability ("drift") detection and characterization on time-series data. This data analysis is circuit-agnostic, i.e., it can be applied to data from any set of quantum circuit over any number of qubits. It consists of looking for evidence that the "probability trajectories" of the circuits, meaning the potentially time-varying probability distributions over the possible circuit outcomes, are varying over time. In the language of hypothesis testing, the null hypothesis is that each circuit has some static (but otherwise arbitrary) probability distribution over its possible outcomes, and we are testing for violations of this null hypothesis.
